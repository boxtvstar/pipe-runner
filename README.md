# Pipe Runner

브라우저에서 바로 실행되는 오리지널 픽셀 플랫폼 게임입니다. 마리오풍의 점프 액션이지만, 캐릭터와 그래픽은 새로 만든 간단한 캔버스 아트입니다.

## 실행 방법

1. 이 폴더를 다른 컴퓨터로 옮깁니다.
2. `index.html` 파일을 더블클릭해서 브라우저로 엽니다.
3. 시작 버튼을 누르면 게임이 시작됩니다.

## 조작

- `←` / `→`: 이동
- `Space`: 점프, 공중에서 한 번 더 누르면 2단 점프, 날개 상태에서는 누르고 있으면 비행
- `F`: 총알 발사
- `R`: 처음부터 다시 시작
- 모바일/태블릿: 왼쪽 조이스틱을 좌우로 밀어서 이동, 오른쪽 `↑` 버튼으로 점프

물음표 박스에서는 코인, 커지는 버섯, 총알팩, 5초 비행 날개가 나옵니다. 큰 상태에서 벽돌을 아래에서 치면 깰 수 있습니다.

## 파일 구성

- `index.html`: 게임 화면
- `styles.css`: UI 스타일
- `game.js`: 게임 로직
